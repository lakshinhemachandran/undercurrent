# Undercurrent

Undercurrent is a teen-focused self-reflection and therapy-tracking app built with Next.js, Tailwind CSS, Supabase, Zustand, and Framer Motion.

Created by Lakshin Hemachandran

## Environment setup

1. Copy `/home/runner/work/undercurrent/undercurrent/.env.example` to `.env.local`.
2. Fill in the required values.
3. Add your OpenRouter key in the blank placeholder:

```env
OPENROUTER_API_KEY= 
```
